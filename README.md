# Story Share

A professional story-sharing web application inspired by popular social media platforms such as Instagram and WhatsApp.

This project demonstrates how to create a client-side story feature using HTML, CSS, and JavaScript. Users can upload images, view stories in a fullscreen viewer, navigate between stories, and store data using Local Storage. Stories automatically expire and are removed after 24 hours.

---

## Features

- Upload Stories from Local Device
- Base64 Image Conversion
- Local Storage Integration
- Instagram-Style Story List
- Fullscreen Story Viewer
- Previous and Next Navigation
- Automatic Story Expiration (24 Hours)
- Responsive Design
- Client-Side Only Application
- Story Persistence After Page Refresh

---

## How to Run

1. Download or clone this repository.

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click on:

   index.html

5. Select:

   Open with Live Server

6. The application will open automatically in your browser.
